# BelowTheLeague Live (V1)

This is the mobile-first live match centre for the Isthmian League (Step 4 North), built using:

- Next.js App Router
- TailwindCSS
- React Components
- Vercel deployment

## Features (V1)
- Live Scores
- Vidiprinter-style live reporting
- Fixtures & Results
- League Table
- Team Pages
- Match Centre (Timeline, Lineups, Stats, Report)

## Folder Structure
- /app → Routes
- /components → UI components
- /components/live → Live screen
- /components/match-centre → Match centre screens
- /components/fixtures → Fixtures/Results screens

This project is designed for fast iteration and mobile-optimised UI based on the BelowTheLeague brand.
